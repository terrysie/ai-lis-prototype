# TERRY-LIS v0.6 Roadmap Scope

## 1. v0.6 定位

v0.6 是试用反馈版 / Demo Guide / 首次打开体验优化版本。

核心目标是让第一次接触 TERRY-LIS 的用户能够快速理解：

- 系统是什么
- 可以解决什么问题
- 当前具备哪些能力
- 如何体验 Demo

---

## 2. v0.6 核心目标

- Demo 使用流程优化
- 首次打开体验优化
- 新用户理解路径优化
- 功能说明完善
- 使用指南完善
- 反馈收集准备

---

## 3. v0.6 不包含内容

- 不接入真实医院 LIS
- 不连接真实检验仪器
- 不处理真实患者数据
- 不进入生产环境
- 不改变数据库核心模型
- 不替代正式医院系统

---

## 4. 当前能力基础（来自 v0.5 RC 1）

已经具备：

- 样本接收流程
- 样本签收确认
- 退样与补采任务
- 检验结果审核
- 危急值闭环
- 报告生成
- 报告发布
- 权限控制
- 敏感操作保护
- 审计记录

---

## 5. v0.6 后续规划方向

- Demo onboarding
- 新用户引导
- 示例数据体验
- 使用文档
- 产品介绍
- 用户反馈收集

---

## 6. v0.6 验收标准

- 新用户能够理解系统定位
- Demo 流程完整
- README 入口清晰
- 文档入口清晰
- Release 下载入口明确
