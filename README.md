# AI-LIS 第一版原型需求

这是一个 AI-LIS 实验室信息管理系统原型，面向单实验室 / 单院区。

第一版先做静态前端原型，不接入真实医院系统，不使用真实患者信息，不需要后端和数据库。

V1 核心展示三件事：

1. 智能预审 / 自动放行
2. 危急值自动闭环
3. 内部智能助手

AI 不替代最终医疗责任人，只做筛选、解释、催办、排序。

第一版页面包括：

- 审核技师工作台
- 样本流转概览
- AI 预审队列
- AI 判定卡片
- 危急值中心
- 内部智能助手

AI 判定卡片必须采用：

1. 结论
2. 命中规则
3. 建议动作

技术要求：

- 使用 HTML、CSS、JavaScript
- 代码先写在 index.html
- 所有数据使用模拟数据
- 保持 GitHub Pages 可部署
